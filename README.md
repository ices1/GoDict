# GoDict

### Desc

Command line tool for translating English words into Chinese.

### install
```js
    npm install godict
```

### run
```js
    GoDict >>> word

    // #  1
    // Cn:  词；字；单词
    // En:  a single unit of language that has meaning and can be spoken or written
    // example:
    //    你的论文不应超过2000词。
    //    Your essay should be no more than two thousand words long.

```
